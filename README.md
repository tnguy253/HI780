# Identifying Lifestyle Risk Factors Associated with Heart Disease Using CDC BRFSS Data

This project applies machine learning techniques to identify lifestyle and clinical factors associated with heart disease using the CDC BRFSS dataset. 

## Team ART
Team Members: Austin Patteson, Rathish Moorthy, Thuy Nguyen

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Tools & Libraries
- Python
- pandas
- numpy
- scikit-learn
- matplotlib
